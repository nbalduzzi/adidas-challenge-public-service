## Description

Adidas Challenge Public Service API

## Considerations

**NodeJS version:** 12

**Framework used:** [NestJS](https://nestjs.com/)

**Documentation**: [Swagger](https://swagger.io/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
# Docker build image
$ docker build . -t adidas-challenge-public-service

# Docker run image
$ docker run -p3000:3000 adidas-challenge-public-service
```

## Documentation

Swagger documentation on dev instance: `http:localhost:3000/api`
